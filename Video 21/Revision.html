Understanding CSS Specificity

What is CSS Specificity?

CSS specificity is a set of rules that determines which styles are applied to an element when there are conflicts. It is calculated based on the types of selectors used.

  • Inline styles (e.g., style="") have the highest specificity.
  • ID selectors are stronger than class selectors.
  • Class, pseudo-class, and attribute selectors are equal in strength.
  • Element and pseudo-element selectors have the lowest specificity.
  • !important overrides everything except other !important rules with higher specificity.
